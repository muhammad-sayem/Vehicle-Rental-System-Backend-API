# Project Name: 
Vehicle Rental System

# Live URL: 
https://vehicle-rental-system-eight-sigma.vercel.app

# Features:
CRUD API creation for a vehicle rental system, where user creation and login system available and also JWT token is implemented for authentication, authorization and role base access.

There are 3 tables "users", "vehicles", "bookings". Here we implemented CRUD through this tables and also did operations with relation between them.

# Teck Stack:
TypeScript, node.js, express.js, postgreSQL, JWT

# Stup & Usage Instructions: 
* Clone the repository
* Open in VS Code
* Open terminal and run "npm install"
* include .env file 
* environment variables:
    PORT=5000
    CONNECTION_STR=*********************
    JWT_SECRET="*****************"
* Run command "npm run dev"
